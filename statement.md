# Statement of Purpose (Project-Based)

As an undergraduate student in aerospace engineering, I have developed a
strong interest in fluid mechanics and its applications in aerodynamics
and flow analysis. While studying these concepts theoretically, I became
curious about how fluid flow is actually simulated in real-world
engineering problems. This curiosity led me to undertake a
beginner-level project combining Computational Fluid Dynamics (CFD) with
basic machine learning techniques.

The objective of this project was to build a simple 2D CFD solver and
use the generated data to train a neural network capable of predicting
the evolution of fluid flow over time. Through this work, I aimed to
gain a foundational understanding of both numerical simulation methods
and data-driven modeling approaches.

During the project, I implemented a basic finite-difference CFD solver
to simulate incompressible flow and generate velocity field data. I then
designed a simple convolutional neural network (CNN) to learn the
mapping between consecutive time steps of the flow. Although the model
was relatively simple, it was able to capture short-term flow behavior
and demonstrate how machine learning can assist in accelerating
simulations.

This project presented several challenges, including setting up the
programming environment, understanding numerical stability, and
improving model performance. Overcoming these difficulties helped me
develop problem-solving skills and a deeper appreciation for both
computational methods and machine learning techniques.

More importantly, this experience helped me bridge the gap between my
core aerospace engineering background and modern computational tools. It
reinforced my interest in exploring interdisciplinary approaches,
particularly in areas where traditional engineering methods can be
enhanced using AI.

Overall, this project represents an important step in my learning
journey. It reflects my initiative to go beyond theoretical study, apply
concepts practically, and explore emerging technologies relevant to
aerospace engineering. I believe it is a meaningful addition to my
academic profile and a strong foundation for further work in simulation
and data-driven engineering methods.
